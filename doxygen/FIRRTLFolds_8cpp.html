Implicitly replace the operand to a constant folding operation with a const 0 in case the operand is non-constant but has a bit width 0.

This makes constant folding significantly easier, as we can simply pass the operands to an operation through this function to appropriately replace any zero-width dynamic values with a constant of value 0.
