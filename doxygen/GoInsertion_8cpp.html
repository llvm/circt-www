Adds the group's "go" signal to the guards of assignments within group, with the exception of the "done" terminator.

If the assignment already has a guard, then the bitwise 'and' is taken of the current guard and the "go" signal. For example:

%go = calyx.group_go %undef : i1
// Case 1: No Guard
%in = %out : i8
=>
%in = %out, %go ? : i8
// Case 2: Guard
%in = %out, %guard ? : i8
=>
%and = comb.and %guard, %go : i1
%in = %out, %and ? : i8
