◆ getFieldID()

unsigned BundleType::getFieldID ( unsigned  index)

Get an integer ID for the field.

Field IDs start at 1, and are assigned to each field in a bundle in a recursive pre-order walk of all fields, visiting all nested bundle fields. A field ID of 0 is used to reference the bundle itself. The ID can be used to uniquely identify any specific field in this bundle.
