Detailed Description

Represents a finite word-length signed integer in SystemC as described in IEEE 1666-2011 §7.6.5.

It is allowed at all places where 'SignedType' is allowed as it inherits and preserved all its functionality as described in IEEE 1666-2011 §7.6.5.1. The difference is that the bit-width has to be passed as a template argument and is thus known at compile-time.
