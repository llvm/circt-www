Detailed Description

Represents a limited word-length signed integer in SystemC as described in IEEE 1666-2011 §7.5.4.

It is allowed at all places where 'IntBaseType' is allowed as it inherits and preserved all its functionality as described in IEEE 1666-2011 §7.5.4.1. The difference is that the bit-width has to be passed as a template argument and is thus known at compile-time.
