Detailed Description

Represents a finite word-length bit vector (of four-state values) in SystemC as described in IEEE 1666-2011 ยง7.9.6.

It is allowed at all places where 'LogicVectorBaseType' is allowed as it inherits and preserved all its functionality as described in IEEE 1666-2011 ยง7.9.6.1. The difference is that the bit-width has to be passed as a template argument and is thus known at compile-time.
